# Femto [![Build Status](https://travis-ci.com/3vilcookie/femto.svg?branch=master)](https://travis-ci.com/3vilcookie/femto)

CLI Text-Editor in Vanila C.

# Usage

Show help:

```bash
$ femto -h
```
Show version: 
```bash
$ femto -v
```

Open existing file:
```bash
$ femto README.md
```

New file:
```bash
$ femto
```
